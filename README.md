# PalindromeCheckerApp

## Running Procedure

1. Compile the Java file using:
   javac PalindromeCheckerApp.java
2. Run the program using:
   java PalindromeCheckerApp
3. Enter the required input when prompted.

## Flow of Project

1. Program starts
2. Displays welcome message
3. Accepts input string from user
4. Processes the string (remove spaces, convert to lowercase)
5. Reverses the string
6. Compares original and reversed string
7. Displays whether it is palindrome or not
8. Program ends

## Objective

To develop a Java application that checks whether a given word or sentence is a palindrome using user input.

## What Are Topics It Will Be Covered

* Java Basics
* String handling
* User input using Scanner
* Conditional statements (if–else)
* StringBuilder / reverse logic
* Program flow control

## Which Usecase

Use Case 2:
The system accepts a word or sentence from the user and checks whether it is a palindrome, ignoring case differences and spaces.

## Working of Use Case 2

1. User runs the application.
2. System asks the user to enter text.
3. The entered text is cleaned (spaces removed, converted to lowercase).
4. The system reverses the cleaned text.
5. If both strings match → displays **Palindrome**.
6. Otherwise → displays **Not a Palindrome**.
